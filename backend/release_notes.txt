POSBah v2.18.3 - Varian Produk, Harga Grosir, Rincian HPP, dan Target Penjualan Outlet

Halo Kak! POSBah v2.18.3 hadir dengan peningkatan fitur komprehensif pada modul POS FnB:
1. Varian Produk POS FnB: Tambahkan pilihan varian seperti coklat, ukuran L, ukuran M secara fleksibel saat input produk, pilih varian saat transaksi POS, dan cetak varian terpilih di struk belanja.
2. Harga Grosir Dinamis: Tetapkan harga grosir dan batas jumlah minimum pembelian. POS otomatis menghitung harga grosir ketika jumlah pembelian di keranjang tercapai.
3. Rincian Komponen HPP: Penambahan komponen HPP produk (misal: Nasi, Lele, Kertas) yang otomatis terakumulasi sebagai Harga Beli produk untuk mempermudah analisis laba kotor.
4. Target Penjualan Harian per Outlet: Pemilik bisnis dapat menetapkan target penjualan harian per produk untuk masing-masing outlet. Target harian bersifat immutable (history aman) dan otomatis terisi dengan template target default.
5. Pembatasan Hak Akses: Proses penambahan dan pengubahan produk kini dibatasi secara ketat hanya dapat dilakukan oleh Owner atau Administrator/Admin.

POSBah v2.18.0 - Peningkatan Performa, Cache Memori & Pembersihan Otomatis

Halo Kak! POSBah v2.18.0 hadir dengan peningkatan performa dan efisiensi memori yang signifikan:
1. Pembersihan Memori HP Otomatis: Aplikasi sekarang otomatis mendeteksi dan menghapus berkas database lama (offline-first) dari memori internal HP Anda setelah migrasi full online untuk membebaskan ruang penyimpanan.
2. Fitur Cache Memori Statis (Read-Only Cache): Menghindari request API berulang-ulang untuk data statis (Outlet, Karyawan, dan Master Produk), sehingga aplikasi menjadi jauh lebih cepat, hemat kuota, dan responsif.
3. Pembaruan Kecepatan (Optimistic UI): Peningkatan respon penginputan data pada POS, Invoice, dan Manufaktur.
4. Gzip Compression: Mengaktifkan kompresi data transfer di server VPS agar loading data berukuran besar menjadi 70% lebih cepat dan irit kuota.
