Pembaruan POSBah Versi 2.0.5

1. Pembaruan Keamanan Masuk:
   Akun Premium sekarang diwajibkan menggunakan tab Masuk Premium (Email & PIN) demi keamanan data. Login Demo (Google SSO) kini dibatasi untuk akun non-premium saja.
2. Fitur Pemulihan Akun (Rejoin):
   Bantuan pemulihan bagi akun terhapus kini dapat diajukan dengan mudah lewat tombol "Hubungi Admin" di halaman masuk untuk memulihkan akun secara resmi melalui konfirmasi email.
3. Peningkatan Kinerja:
   Sinkronisasi database serta koneksi real-time antara aplikasi kasir dan server berjalan lebih lancar dan aman.
